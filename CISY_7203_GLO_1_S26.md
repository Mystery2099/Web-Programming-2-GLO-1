# CISY 7203

## Graded Learning Opportunity #1

This GLO is due in one week by the beginning of your class.

This assignment requires you to create a website that celebrates the joys of March by detailing information about all the fun you and other students can have during this important month. This assignment is broken down into several components.

1. Create a strict web site structure to contain your required material. Please title the starting page index. This page should also serve as the interface to the rest of the work you create for this GLO. This site structure must include you demonstrating the use of:
   - Images
   - Text
   - Examples of common HTML Structural elements
     - Tables, headings, etc.
   - Navigation to the pages you create for assignments 2, and 3
   - Commonly accepted development and implementation principles
   - CSS formatting including the use of:
     - Inline, internal and external styles
     - Backgrounds
     - Margins
     - Floating elements
     - Borders
     - Font manipulations
   - JavaScript for interactive enhancements
     - At least three formatting changes
     - Interactive processing
       - (I.E. validation, input management, input manipulation, etc.)

2. Data bind information about yourself, March holidays and general tips to enjoy them safely.
   - Data bind information about you and your plans for March.
   - Data bind information that lists many of the common March holidays and events
   - Data bind information regarding how to enjoy the month safely.
   - These documents must be well formed
   - Include at least ten records within your data bound information (its okay to make some things up, if you are miserable and dislike thinking or if you just like being creative.)
   - Be sure to format your data bound objects appropriately.

3. Be sure that your web pages demonstrate appropriate design and accessibility.
   - Mobile accessibility
   - Follow the principles of interactive design
