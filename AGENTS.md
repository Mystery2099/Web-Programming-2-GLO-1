## IMPORTANT: Dev Server Policy

**NEVER RUN THE DEV SERVER.**

The user typically runs the development server themselves. Starting the server is annoying and interferes with their workflow.

Do NOT run any of these commands:

- `bun dev`
- `bun run src/server/index.js`
- Any command that starts a background server process

The user will handle starting and stopping the dev server as needed.
